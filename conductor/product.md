# Product Guide: ruttu-dictation

## Initial Concept
ruttu.ee - Fast Dictation for macOS.

## Product Vision
ruttu-dictation is a high-performance, real-time dictation tool designed for macOS. It bridges the gap between thought and text by leveraging the Deepgram API for low-latency transcription. The application lives in the menu bar, providing instant access to voice-to-text capabilities across any application. It is designed to be lightweight, unobtrusive, and highly accurate, supporting multiple languages with a focus on Estonian, Russian, and English.

## Target Audience
The application caters to a diverse group of users who value speed and efficiency:
*   **Professional Writers & Journalists:** Individuals who need to draft content quickly and capture ideas as they flow.
*   **Developers:** Professionals seeking hands-free methods for writing documentation, code comments, or communicating in chat.
*   **Multilingual Users:** Specifically those in Estonia or similar regions who frequently communicate in a mix of Estonian, Russian, and English.
*   **General Users:** Anyone looking to reduce typing strain or speed up their daily text entry tasks.

## Core Value Proposition
*   **Speed:** Instant transcription with minimal latency using Deepgram Nova-3.
*   **Accessibility:** Always-available menu bar integration.
*   **Versatility:** Smart formatting, automatic punctuation, and seamless multi-language support.
*   **Privacy & Control:** Local voice activity detection (VAD) and configurable hallucination filtering.

## Key Features
*   **Real-time Transcription:** Streaming audio to text insertion.
*   **Global Hotkey/Menu Bar Control:** easy activation and status visualization.
*   **Smart Typist:** Automatic handling of punctuation and formatting.
*   **Voice Commands:** Support for spoken commands like "new line" or "bullet point".
*   **Customizable Settings:** API key management, language selection, and custom exclusion filters.
